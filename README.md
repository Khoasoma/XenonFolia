## Build
Để Build được paperclip jar dùng cmd dưới đây. Build xong sẽ tìm thấy jar ở build/libs (Note: Cần jdk21)
 ```shell
 ./gradlew applyPatches && ./gradlew createMojmapPaperclipJar
```
